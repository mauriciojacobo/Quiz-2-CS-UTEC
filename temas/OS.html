Sistemas operativos

¿Qué es un sistema operativo? ¿Cuales son sus funciones?

Definición

Es aquel programa que se encarga de manejar la interacción entre el softaware con el hardware
para poder facilitar la experiencia usuario o la eficiencia de recursos

Gestión de la memoría

En general el sistema operativo es el encargado de gestionar
la mayoría de procesos del hardware, pero el manejo de memoría tiene
más de relevancia, ya que depende de este cuan eficiente puede ser el procesador o CPU

Seguridad

La seguridad también es considerada una función de un sistema operativo ya que dependiendo de este se determina cuan seguros son los procesos que corren durante el uso del computador

Interfaz de usuario

Más inclinado hacia la diferenciación de producto los distintos sistema operativos tratan de tener una interfaz grafica unica (Ubuntu, Windows, MacOS, etc), esto para los usuarios finales suele ser de gran importancia
